# BD-Datetime-Picker

BD-Datetime-Picker is a react component for datetime picker redoing from [rc-datetime-picker] for my needs.
Thanks [AllenWooooo]!!!

## Requirements

- React
- [Moment.js]
- Modern browsers (IE>=9 is required)

## Installation

### Install with NPM

```
$ npm install bd-datetime-picker
```

### Install with yarn

```
$ yarn add bd-datetime-picker
```
